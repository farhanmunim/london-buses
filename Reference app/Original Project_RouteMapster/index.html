About RouteMapster

About

RouteMapster is a unified London bus network explorer, built to explore coverage, operators, and service patterns quickly and intuitively.

Data sources

  • TfL Bus Open Data - route geometries
  • TfL Unified API - bus stops and timetable data
  • londonbusroutes.net - garage allocations
  • postcodes.io - postcode data
  • OpenStreetMap - map
